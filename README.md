Static one-page waitlist site for a babywearing carrier library in the Willamette Valley, OR.

## One-click deploy (Vercel)

1. Push this folder to a Git repo.
2. In Vercel, click **New Project** and import the repo.
3. Framework preset: **Other** (no build command).
4. Output directory: `.` (project root).

Once deployed, set the custom domain to `library.babysbreathbw.com`.

## Email form

The form posts to FormSubmit and sends submissions to `hannah@babysbreathbw.com`.

## Local preview

Open `index.html` directly in a browser or use a static server.
